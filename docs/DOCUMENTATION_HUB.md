### ☸ Project Overview

The Dhammapada Study Companion is a professional-grade platform designed to bridge ancient wisdom with modern technology. It features:
- **Canonical Accuracy**: Root text and translations from SuttaCentral.
- **Intelligent Search**: Semantic and lexical hybrid search engine.
- **AI Tutoring**: A context-aware chatbot for scriptural study.
- **Scholarly Integrity**: Transparent source attribution and word-by-word analysis.
